Acordeón Básico

¿Qué es lucen UI?

lucen UI es un framework complementario para Bootstrap 5 que proporciona componentes con estilo Fluent Design (Windows 11), usando solo HTML, CSS y JS.

¿Cómo se usa?

Simplemente incluye los archivos CSS y JS de lucen UI en tu proyecto después de Bootstrap 5. Todos los componentes están listos para usar con las clases proporcionadas.

Personalización

Puedes personalizar los colores, tamaños y otros aspectos mediante las variables CSS proporcionadas en el archivo principal.

Acordeón Fluent (Acrílico)

Características Principales
  • Diseño basado en Fluent UI
  • Efectos acrílicos
  • Animaciones fluidas
  • Optimizado para touch
  • Compatibilidad con Bootstrap 5
